# 🎮 Personagem e Colisão na Unity

### 📌 Descrição

Este projeto foi desenvolvido como exercício acadêmico para praticar conceitos básicos de **criação de personagens e detecção de colisões** na Unity.
O objetivo principal é adicionar obstáculos na cena e implementar uma lógica que altera a cor dos objetos quando ocorre colisão com o personagem.

---
### 🛠️ Funcionalidades

- Criação de um novo projeto na Unity.

- Inclusão de um personagem jogável em um ambiente 3D.

- Adição de múltiplos obstáculos (ex.: Cubes).

- Alteração da cor dos objetos ao detectar colisão com o personagem.
---

### 🚀 COMO EXECUTAR
1- Clone este repositório:
```
git clone https://github.com/seu-usuario/Personagem-e-Colisao.git
```
2- Abra o projeto na Unity (versão recomendada: 2021.3 LTS ou superior)

3- Execute a cena principal (SampleScene) para testar o personagem e as colisões
